LED Mural Board

  • Category: Mechanisms/Art/EE
  • Tools Used: Microcontroller, Soldering, C
  • Project date: February 9, 2024

Project Description

During MIT's 2024 independent winter activities session (IAP), I spent some time making an LED Board with some friends. The goal was to create a mural board using LEDS that could showcase art through an unconventional medium. We applied for and were accepted into ProjX, a project fund, which granted us $500 for materials including LEDs, wood paneling, copper sheet, and wire. We attached 1500 LEDs to a wooden paneling, soldered them together, and controlled them via an arduino programmed with the FastLED library. At the end of the winter session, we presented this at the ProjX Expo alongside some of the other participants. Hopefully in the future, we can continue this project by programming more advanced visuals onto the LEDs, but for the Expo, we had some pretty rainbows to show.
