LED Mural Board

Category: Art / EE

Tools Used: Microcontroller, Soldering, C

Project date: February 9, 2024

Project Description

During MIT's 2024 Independent Activities Period (IAP), I worked with friends to create a mural board that showcased art through LEDs. We were accepted into ProjX, which granted us $500 for materials.

We mounted 1500 LEDs on wooden paneling, soldered them together, and programmed the system with an Arduino using the FastLED library. At the ProjX Expo, we demonstrated the board with rainbow effects. Future plans include more advanced visual programming.
